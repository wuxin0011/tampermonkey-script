


# 弹幕屏蔽插件


按下 ctrl + alt + k 即可生效，对于不想看关键词可以自己填写 如 `6666` 、`233333`, `xxxx驾临直播间`  等等

 - [虎牙](https://www.huya.com)
 - [斗鱼](https://www.douyu.com)
 - [B站](https://live.bilibili.com)
